This project includes a simple relational database for managing players, teams, trainers, and trainer education in a sports club.

Files included:
- club_management_UML.drawio: ER diagram showing entities and relationships with correct cardinalities.
- SQL script: Tables and sample data created in MySQL.
- CSV files: Results of 5 queries according to the assignment instructions.

Queries cover:
1. Listing all players with their team (sorted by player name).
2. Listing all trainers grouped by their education.
3. Showing each player with their corresponding trainer.
4. Counting the number of players per trainer.
5. Increasing training time by 25% for a trainer.
